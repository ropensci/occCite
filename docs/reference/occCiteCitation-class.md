# occCite Citation Class

A class for managing citations generated from occCite queries.

## Fields

- `occCitationResults`:

  The results of performing
  [`occCitation`](https://docs.ropensci.org/occCite/reference/occCitation.md)
  on a
  [`occCiteData`](https://docs.ropensci.org/occCite/reference/occCiteData-class.md)
  object, stored as a named list, each of the items named after a
  searched taxon and containing a data frame with occurrence
  information.
